# 🧪 Prompt Lab (프롬프트 랩)

**"생성형 AI 프롬프트 학습의 모든 것"**  
Prompt Lab은 다양한 환경(비즈니스, 공공기관, 교육 등)에 맞춘 실전 프롬프트를 학습하고, AI를 통해 자동으로 생성하며, 체계적으로 관리할 수 있는 **올인원 프롬프트 플랫폼**입니다.

---

## ✨ 핵심 기능

### 1. 🤖 스마트한 프롬프트 생성
- **이미지 분석**: 이미지를 업로드하면 AI(Gemini 1.5)가 상황에 맞는 프롬프트를 자동으로 제안합니다.
- **다양한 모드**: 텍스트 주제 입력부터 대량 일괄 생성까지, 클릭 한 번으로 DB를 구축하세요.

### 2. 📚 단계별 맞춤 실습
- **대상별 커리큘럼**: 비즈니스, 공공기관, 학교(초·중·고·대), 일반인 등 타겟 맞춤형 카테고리를 제공합니다.
- **레벨 시스템**: `초급` → `중급` → `고급` 단계별 학습으로 AI 활용 능력을 체계적으로 키울 수 있습니다.

### 3. 🛡️ 강력한 관리 시스템
- **대시보드**: 그룹(계정) 및 프롬프트 데이터를 한눈에 관리합니다.
- **연속 작성(Thread)**: 상세 화면에서 문맥을 유지하며 꼬리에 꼬리를 무는 프롬프트를 작성할 수 있습니다.

---

## 🛠️ 기술 스택
- **Framework**: Next.js 14+ (App Router)
- **Database**: Supabase (PostgreSQL)
- **AI**: Google Gemini Pro & Flash
- **Deployment**: Vercel

---

## 🚀 시작하기

### 1. 설치
```bash
npm install
```

### 2. 환경 설정 (.env.local)
```env
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
GEMINI_API_KEY=...
```

### 3. 실행
```bash
npm run dev
```
